Using a Virtual DOM, we can utilize a model architecture for better performance.

Example Poster Card (LEFT Orientation + Polling)

Simply tag your div with .publicity-card class and your ad will be filled and rendered automatically.

You can even turn on Polling and receive a new card every 15 seconds to keep users engaged.

3x5 Arrangement (Grid)

Creating a set of poster cards is as easy as specifying the rows and columns you want using data-* attributes.

The matrix size will be delivered to screens that support it, for different devices and resolutions the cards will adjust automatically.

HTML Snippet — 4x3 Arrangement + Lazy-loading 

                    <!-- An Arrangement: 4x3 -->
                    <div id="example_grid" class="publicity-arrangement" data-rows="4" data-cols="3"></div>
Javascript/JSX Snippet — Dynamic Arrangements


                    /**
                    * Poster Arrangement (Ad Grid)
                    * -- an arrangement consists of one or more poster cards

                    render: function () {
                    var posterCards = this.props.data.map(function (card, cid) {
                    return (
                    <Publicity.PosterCard key={ cid } />

                    });
                    return (
                    <div className="card-arrangement">
                    {posterCards}
                    </div>
                    );
                    }
